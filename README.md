# Features

・他のデバイスでも確認できるよう共有機能付き  
・日付表示  
・合計と残りの ToDo 数を表記  
・飽きないかつ柔らかい雰囲気のシンプルな UI デザイン  
・ボタンのどこを触れてもチェックのつけ外しが可能  
・名前の変更が可能
